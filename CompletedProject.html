Login App Fit Trace

This app was developed as the final project for Mobile Development. It includes key features such as a login screen, the ability to create routines for better traceability, user preferences like light and dark mode, and integration with an API to provide valuable information about workouts and techniques.

Create Routine Page

One of the core features of this mobile app is the ability to create customized workout routines tailored to the user's goals, preferences, and fitness level. The interface allows users to select exercises by muscle group—such as shoulders, as shown in the image—and assign them to different routine slots. Each dropdown menu corresponds to a specific exercise list, making it easy to build structured sessions
